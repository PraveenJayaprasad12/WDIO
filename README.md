# WebdriverIO Test Repository

This repository is set up with WebdriverIO, Mocha, and ChromeDriver for browser automation testing.

## Getting Started

### Install dependencies
```
npm install
```

### Run tests
```
npx wdio run wdio.conf.js
```

## Example Test
- See `test/google.spec.js` for a sample test that opens Google and checks the page title.

## Configuration
- The main configuration file is `wdio.conf.js`.

---
For more information, visit the [WebdriverIO documentation](https://webdriver.io/docs/gettingstarted/).
